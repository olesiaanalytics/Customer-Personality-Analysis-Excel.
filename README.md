# Marketing Data Analysis | Customer Personality Case Study
### What I did in this project
I worked with a raw marketing dataset to see if I could find any clear patterns in how different customers spend their money. It wasn't just about making charts; I wanted to see the logic behind the numbers.

Cleaning the mess: The data had some missing values in "Income" and "Year_Birth", so I cleaned those up first. I also found some extreme outliers that were completely throwing off my trendlines, so I had to remove them to get a realistic picture.

Creating new categories: I calculated customer ages and grouped them into segments like "Seniors" or "Adults". I also added a "Customer Status" column (Active vs. Lapsed) and checked how many kids they have, just to see if that changes anything.

Building the Dashboard: I set up several Pivot Tables to see who buys what. To make it easy to play around with the data, I added Slicers. Now you can just click a button (like "Seniors" or "Parents") and the whole dashboard updates instantly.

Finding the link: I used the CORREL function to check the relationship between income and premium products. It turns out there’s a strong 0.7 correlation, which I visualized with Scatter Plots.

### My main findings
Seniors are the biggest spenders: They buy twice as much wine as any other group and are very active both in-store and online.

Wine and Meat are the stars: These two categories bring in the most revenue by far.

Income matters: The correlation confirmed that the more people earn, the more they spend on premium items, which seems obvious but it’s good to see it backed by actual data.
